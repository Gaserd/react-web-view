react-web-view
==========================


How works
-------------

```
      <ReactWebView
        url={'https://facebook.github.io/react/'}
        navigation_text={'React'}
        animation={'bottomontop'}
       />
```

Url(*) - parametr of your url page

navigation_text - text of navigation bar

animation - animation of iframe
